How to play

  1. Goal: Fill the 9x9 grid so every row, column,
    and 3x3 sector contains numbers 1-9.
  2. Rules:
    • Each number 1-9 appears once per row.
    • Each number 1-9 appears once per column.
    • Each number 1-9 appears once per 3x3 box.
  3. Start: Some numbers are already placed.
    Use them as clues.
  4. Play:
    • Use intersections of rows, columns, and boxes to exclude impossible numbers.
    • Use logic (not guessing) to fill empty cells.
  5. Win: The puzzle is solved once all cells are filled correctly.
